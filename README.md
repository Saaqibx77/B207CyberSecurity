# Secure Password Manager

## Project Description

This is a simple Secure Password Manager developed using Python, Flask and SQLite.

The project allows users to:
- Register a new account
- Login securely
- Generate strong passwords
- Check password strength
- Save website credentials
- View saved passwords
- Logout safely

## Technologies Used

- Python
- Flask
- SQLite
- HTML
- CSS
- hashlib

## Installation

Install Flask

```
pip install -r requirements.txt
```

Run the project

```
python app.py
```

Open your browser

```
http://127.0.0.1:5000
```
